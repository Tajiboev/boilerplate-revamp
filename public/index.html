Boilerplate Revamp

Automating simple HTML development workflow with Gulp.js

How to use the boilerplate?

  1. Download from GitHub
  2. Install Node Modules:
    npm install
  3. (Development) Run:
    gulp
  4. (Build) Run:
    gulp build

Gulp tasks:

TaskDescription
"default"Runs 'styles' and starts a server. Watches changes in .html, .sass and .js files
"html"Minifies the .html file
"serve:src"Starts a server using browser-sync
"styles"Compile .sass files to .css (automatically prefix and minify)
"scripts"Transpile, concat and uglify .js files
"images"Resize all .jpg files in images/ to 800, 1280 and 1920 pixels in .jpg & .webp formats
